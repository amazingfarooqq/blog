What is an API? Real Life Example:

You are hungry and need something to eat

You could go to Store, buy required grocery and cook it yourself but as you see, you have to do alot of thing's to accomplish it

Second option is that you can get it from the restaurant

Professional chefs will already have all the resources, but the chef don't want you to access all his stuff due security and safety purpose,

That is where an intermediate is needed to provide allowed services from one place to another......

That serving guy,( in another words 'API' ) who serve data from one place to another without letting us get into kichen where chef is making all his stuffs

API stands for application programming interface, it is a software intermediary that allows twoapplications to talk to each other.

One common application of API is Sign up on different sites with already created Gmail or Facebook, your data is already saved at one place and the app you use takes data from Gmail/Facebook as an API.

API's are needed to bring applications together in order to perform a designed build around sharing data and executing pre-defined processes, Few popular API's is Google Maps
